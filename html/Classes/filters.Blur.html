fabric. Image. filters. Blur

模糊滤镜类 继承fabric.Image.filters.BaseFilter

var filter = new fabric.Image.filters.Blur({
    blur: 0.5
});
object.filters.push(filter);
object.applyFilters();
canvas.renderAll();

成员

blur

模糊值，以图像尺寸的百分比表示。
专门用于在0和1之间的不同分辨率范围内保持图像模糊恒定。

type

滤镜类型

赋值类型： BaseFilter
